# E-commerce Recommendation System (Python)

A simple and effective product recommendation system built using Python.  
This project uses **Cosine Similarity** to recommend similar products based on user ratings.

---

## Features
- Product recommendation system
- Similarity calculation using Machine Learning
- Displays product details (price & category)
- Shows trending products based on ratings
- Category-wise product grouping

---

## How It Works
- User-product rating matrix is created  
- Cosine similarity is applied to find similar products  
- Top similar products are recommended  

---

## Technologies Used
- Python  
- Pandas  
- Scikit-learn
